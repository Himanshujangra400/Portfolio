# 🚀 Himanshu Jangra — Portfolio

A sleek, animated single-page portfolio website built with pure **HTML, CSS & JavaScript**.

## ✨ Features
- Animated hero section with typewriter effect & particle network background
- Smooth scroll-triggered animations (AOS)
- Animated skill progress bars
- Featured project cards with hover effects
- Contact form with mailto fallback
- Fully responsive (mobile-friendly)
- Dark Tokyo Night theme

## 📂 Files
| File | Description |
|------|-------------|
| `index.html` | Main portfolio page |
| `style.css`  | All styles & animations |
| `script.js`  | Particles, typing effect, counters, skill bars |

## 🔗 Live
Deploy via **GitHub Pages**: Settings → Pages → Branch: `main` → `/ (root)`
